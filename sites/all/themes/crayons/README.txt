REQUIREMENTS
----------------------
Node : v0.10.40
Npm : v1.4.28

User NVM(Node Version Manager) to easily switch versions



Installation
----------------------
1. bower install
2. npm install / sudo npm install



Notes
----------------------
Development: 
  
	- Type "grunt" or "grunt watch" compile.
  
  - Type "grunt build" to compile themes and split each theme into 3 css files for IE9.

  Javascript 

  	- adding new script: go inside js/modules and add new file. Grunt will take care of compiling them into one file."
